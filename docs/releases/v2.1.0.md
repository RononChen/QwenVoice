# Vocello 2.1.0

Released **2026-06-12**. Vocello 2.1.0 is a quality-and-performance release for the macOS 26-native, on-device voice generation app. Everything still runs locally on Apple Silicon via MLX; no audio leaves the Mac unless you export it.

> **Upgrading from 2.0.0?** Download the new DMG and drag `Vocello.app` to `/Applications` (replacing the old copy). Your installed models, history, and saved voices live in `~/Library/Application Support/QwenVoice/` and carry over automatically — no re-download needed.

[Download v2.1.0](https://gitlab.com/VocelloApp/QwenVoice/-/releases/v2.1.0) · [All releases](https://gitlab.com/VocelloApp/QwenVoice/-/releases)

## Requirements

- macOS 26.0 or later
- Apple Silicon Mac
- **8 GB RAM minimum** (16 GB+ recommended for Quality variants and heavy batches)
- ~7 GB free disk for the three recommended Speed (4-bit) model variants; ~16 GB if you also install Quality (8-bit)
- Internet for one-time model downloads from Hugging Face

## Headline themes

- **8 GB Macs now generate faster than realtime** with Speed models (RTF past 1.0 on the floor-tier baseline).
- **Voice library grew up** — record references with the Mac microphone, on-device transcription in all 10 languages, language-aware picker recommendations.
- **Delivery and language accuracy** — preset rewrites, sampler-order fix, Latin-script auto-language fix, clone first-word fix.
- **Reproducible generations** — variation control (Expressive / Balanced / Consistent), deterministic seeds, shared seed across batch lines.
- **New `vocello` CLI** — headless generation, batch, voices, models, and deterministic bench harness (built from source, not shipped in the DMG).

## Performance and memory (8 GB Macs)

- Warm-admission gating defers proactive warm-ups when memory is tight; user-initiated generations are never gated.
- Engine retirement on idle reclaims memory on pressured 8 GB Macs; relaunch is transparent on the next generation.
- **Behavior change:** automatic Quality→Speed fallback from 2.0.0 was removed. Quality always loads the genuine 8-bit model or surfaces a real error.

## UI and workflow

- Dark mode only (refined dark appearance pinned).
- Detected-language selector (`Language · Auto`), one-row configuration, Voice Design brief editor polish.
- History clear-all with keep-files vs delete-audio choice; cancel no longer flashes error state.
- Reduce Motion honored on macOS.

## Known limitations

- **macOS 15:** stay on [v1.2.3](https://gitlab.com/VocelloApp/QwenVoice/-/releases/v1.2.3). No 2.x backport planned.
- **iPhone:** on-device engine works in development; App Store / TestFlight distribution is still in progress.
- **Voice Cloning:** only clone voices you own or have permission to use.

## Install

1. Download `Vocello-macos26.dmg` from the [v2.1.0 release](https://gitlab.com/VocelloApp/QwenVoice/-/releases/v2.1.0).
2. Open the DMG and drag `Vocello.app` to `/Applications`.
3. Open Vocello → **Settings → Model downloads** → install the recommended Speed packages (~7 GB total).
4. Generate from Custom Voice, Voice Design, or Voice Cloning.

The DMG is Developer ID signed and notarized with a stapled ticket. `release-metadata.txt` on the release records build provenance.
