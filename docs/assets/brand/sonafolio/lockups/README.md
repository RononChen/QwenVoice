## 当前推荐

- 应用内深色界面、演示和未来 Dock 图标方向：Midnight Paper；
- 文档、App Store 元数据页、浅色背景：Warm Editorial；
- 中文辅助名放在英文主字标下方，宽度约为英文主字标的五分之一，保持品牌层级。

此处仍是品牌组合候选，不是 macOS App 图标。平台圆角容器、图标安全区、阴影和
Liquid Glass 层次应在下一阶段单独制作，不能写回品牌主母版。
