# Sonafolio 中文辅助名称

> 状态：候选 v0.1  
> 简体：`声卷`  
> 繁体：`聲卷`

中文名称使用 Noto Sans SC Medium 与 Noto Sans TC Medium 的官方 Google Fonts 子集生成，
随后转换成自包含 SVG 轮廓。运行时不依赖字体文件。

中文只作为 `Sonafolio` 的辅助显示名，不替代英文主品牌，也不单独作为 App 图标。

## 文件

- `sonafolio-name-zh-hans.svg`：简体中文轮廓；
- `sonafolio-name-zh-hant.svg`：繁体中文轮廓；
- 对应 PNG：暖纸白背景评审图；
- `LICENSE-NotoSansCJK.md`：SIL Open Font License 1.1。
